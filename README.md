# Teslo Api

1. Clonar proyecto
2. ```npm i```
3. Clonar el arcivo ```.env.template``` y renombrarlo a ```.env```
4. Cambiar las varibales de entorno
5. Lecantar la DB
  ```
    docker-compose up -d
  ```
6. Ejecutar SEED
   ```
    http://localhost:3000/api/seed
   ```
7. Levantar con ```npm run start:dev```
